# CoinFloat 隐私政策

最后更新日期：2025-11-01

CoinFloat 是一款用于展示 Solana 代币实时行情的 Chrome 扩展。我们重视用户隐私，并致力于遵循最小权限原则。本文说明扩展在运行过程中处理数据的方式。

## 我们收集或存储的数据

- **本地首选项**：扩展使用 `chrome.storage` 在用户浏览器本地保存代币监控列表、悬浮窗位置、是否显示悬浮窗等设置。此数据仅存储在用户设备，不会同步至我们的服务器。
- **行情数据请求**：扩展通过 HTTPS 向 Dexscreener API (`https://api.dexscreener.com`) 请求公开的市场数据，请求内容仅包含代币或交易对 ID，不包含用户身份信息。

扩展不会收集姓名、邮箱、位置、浏览历史、钱包地址等任何个人可识别信息，也不会记录键盘输入或页面内容。

## 数据的使用方式

- 本地首选项仅用于恢复用户自定义设置，提升使用体验。
- 从 Dexscreener 获取的行情数据仅用于在界面上显示实时价格与 24 小时涨跌幅，不会用于其他目的。

## 数据共享

我们不会出售、出租或共享任何用户数据给第三方。所有本地存储的数据均保留在用户设备中。

## 权限说明

- `storage`：用于保存用户的代币列表和界面设置。
- `alarms`：用于在后台定时触发行情刷新任务。
- `https://api.dexscreener.com/*`：用于访问公开行情接口。

这些权限仅为了实现扩展的单一功能，不会用于其他用途。

## 安全性

- 扩展不执行远程 JavaScript、Wasm 或动态注入脚本，只请求公开的 JSON 数据。
- 所有外部请求均通过 HTTPS 发送，以减少被篡改的风险。

## 政策更新

如未来更新隐私政策，我们将在此文件说明最新版的生效日期。继续使用扩展即表示您同意更新后的条款。
